# SOB Luxe — Premium E-Commerce Storefront

A high-performance, minimalist luxury e-commerce frontend built with modern web technologies. Designed with an editorial aesthetic, featuring buttery-smooth animations, a global state engine, and an enterprise-grade decoupled architecture.

![SOB Luxe Preview](/public/sob-logo.jpg)

## 🚀 Tech Stack

* **Core:** React 18, Vite, TypeScript
* **Styling:** Tailwind CSS v4 (Custom variables & luxurious typography)
* **Animations:** Framer Motion, Lenis (Smooth Scroll)
* **State Management:** Redux Toolkit (Thunks for async operations)
* **Routing:** React Router v6
* **UI Components:** Lucide React (Icons), Sonner (Toasts)

## ✨ Key Features

* **Contract-Driven Architecture:** Frontend is completely decoupled with a mock Service Layer (`api.ts`), ready for plug-and-play backend integration.
* **Premium Animations:** Custom full-screen splash onboarding, sticky scroll galleries, and smooth layout transitions using `AnimatePresence`.
* **State Engine:** Redux handles complex shopping cart logic, user authentication state, and dynamic product filtering.
* **Command Palette:** Apple Spotlight-style global search overlay with instant filtering.
* **User Portal:** A dynamic, tab-based dashboard for managing order history, address books, and payment methods without page reloads.
